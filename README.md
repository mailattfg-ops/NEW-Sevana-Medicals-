# Pharmacy Application

This is a modern pharmacy application built with React, Vite, and Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)
- npm

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Build

To build the application for production:

```bash
npm run build
```

## Deployment

### Vercel

This project is configured for deployment on Vercel. 
The `vercel.json` file handles the rewrite rules for Single Page Application (SPA) routing.

Simply import the repository into Vercel, and it should auto-detect the Vite settings.
